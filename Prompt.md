# SshUmdLoader
Это промпт для создания приложения SshUmdLoaderРоль:

## Task
Твоя задача — помочь мне спроектировать и реализовать CLI-утилиту SshUmdLoader на языке С++.
Описание приложения находится в DesignDocument.md

## Rules
Действуй как Senior Developer. 
Используй архитектурный стиль: Clean Architecture. Ядро бизнес-логики (построение путей, фильтрация дат и тикеров) должно быть строго изолировано от инфраструктуры (SSH-клиент, парсер CLI, чтение конфигурации).
Асинхронность: Скачивание должно быть неблокирующим (non-blocking). Предложи подходящую C/C++ библиотеку для SSH (например, libssh или libssh2) и оберни ее в удобный интерфейс.
Программа должна загружать файлы без создания потоков ОС.
Параллельная загрузка строго ограничена параметром MaxConcurrentDownloads.
Обязателен вывод прогресс-бара в консоль для каждого скачиваемого файла.

Архитектурный задел (Future-proofing): В первой версии скачиваем всё безусловно (создаем/перезаписываем). Однако архитектура должна быть спроектирована так, чтобы в следующей итерации (v2) можно было легко добавить реализацию интерфейса, который проверяет локальное состояние (размер скачанного файла) и выполняет докачку (resume) без изменения бизнес-логики ядра.

- Шаг 3: Ядро и Инфраструктура. Реализуй логику парсинга параметров, обхода директорий и неблокирующего скачивания.
